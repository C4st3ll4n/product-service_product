# Serviço de produto

Esse serviço é um pedaço do ecosistema de micro serviços de Gerenciamento de Inventário/Produtos.
Sendo esse o serviço de produto/vitrine

### Dependências
* Spring (boot)
* Micrometer
* Zipkin
* Eureka Client
* testcontainers

É necessário que tenha um server Mongo rodando

### Como rodar
`docker compose up`
